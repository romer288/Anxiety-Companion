Download Options

You can download the code for the Anxiety Companion application using one of the following methods:

1. Download from this Replit

Use the download script to create and download a tarball of the code:

sh download-code.sh

2. Manual Download via Terminal

Run this command in your terminal to create a tarball of the project code:

cd /home/runner && tar --exclude='./node_modules' --exclude='./.git' --exclude='./.cache' -zcf anxiety-companion.tar.gz .
